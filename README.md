# POIS-Lab-1

## Install

```bash
git clone https://github.com/kitanoyoru/POIS-Lab-1.git
```


### Using *poetry* tool

```bash
cd lab1 && poetry install

```

## Run

### Using *poetry* tool
```bash
cd lab1 && poetry run
```



## Dependencies

- [poetry](https://github.com/python-poetry/poetry)
- [flask](https://github.com/pallets/flask)
- [flask-restful](https://github.com/pallets/flask)
